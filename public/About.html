About This Project

Weather Dashboard is a web application that allows users to search weather information for any city worldwide. It uses the OpenWeather API to fetch real-time weather data and displays it in a user-friendly interface.

Features

  • Search weather by city
  • 5 Day Weather Forecast
  • Temperature Charts
  • Current Location Weather
  • Recent Search History

Technologies Used

  • HTML
  • CSS
  • JavaScript
  • Tailwind CSS
  • OpenWeather API
  • Chart.js
